# My Website

This is the README for my website. The website is built using basic HTML, and CSS to show and demonstrate what I have learned in many topics including web development, personal interests, and programming.

**Live Site:** [https://my-website-flame-six-16.vercel.app/](https://my-website-flame-six-16.vercel.app/)

# Tech used:

- Semantic HTML5, CSS - including CSS Flexbox.
- Google Fonts for the [Inter](https://fonts.google.com/specimen/Inter)
- Brand icons from Font Awesome
- Built and hosted on Vercel, connected to the GitHub repository.

# Files:

- index.html # Main landing page
- about.html # About me page
- blog.html # A place with blogs I will write in the future
- contact.html # A list of places you can contact me with links
- styles.css # Core layout, transitions, colors
- README.md # Documentation (this file)

# AI Disclaimer :

I wrote the HTML structure and core styling myself. I used Gemini purely as a tool to help troubleshoot a cross browser double scrollbar bug (turns out I just had my localhost on 67% zoom) and format the aligned Flexbox/gap spacing for the contact links.

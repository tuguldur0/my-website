# "Portfolio" Website

This is the source code of my personal portfolio website! This project is built using vanilla HTML and custom CSS to showcase my development work, personal interests, and community initiatives.

**Live Site:** [https://my-website-flame-six-16.vercel.app/](https://my-website-flame-six-16.vercel.app/)

---

## Tech Stack & Features

- **Frontend:** Semantic HTML5, Custom CSS3 Variables, and CSS Flexbox.
- **Typography:** [Inter](https://fonts.google.com/specimen/Inter) via Google Fonts for a clean, modern geometric sans-serif aesthetic.
- **Iconography:** Font Awesome vector brand icons (Instagram, LinkedIn).
- **Hosting & Deployment:** Automatically built and hosted on **Vercel** connected to the GitHub repository.
- **Design Details:** Interactive hover transitions, capsule navigation layout, and dynamic content cards.
